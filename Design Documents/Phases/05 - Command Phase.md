# 4. Command Phase (§10.1)

The first phase of **each Season** (Spring, Summer, Fall — and a reduced version
in Winter). Players secretly commit a Command card (or Pass) to determine **who
moves this Season, and in what order** (Command Order).

## Sequence

1. Rotating in **Turn Order**, each player either **commits** a Command card
   face-down or **Passes**.
2. When **all players Pass in succession**, reveal all committed cards.
3. Determine which players take Player Turns and in what **Command Order**.
4. If no cards were committed at all, **advance to the next Season**.

---

## 10.1 Committing Commands

- Players must **Pass** if they have already committed a card (you commit at most
  one card per Season).
- **Action cards** are played sideways for Command purposes. Each is valid for a
  specified **Season** (Spring/Summer/Fall) and shows:
  - a **Command Priority** (a letter), and
  - a **Command Value** (a number — how many units may move, §11.0).
- The player with the **earliest Command Priority letter** takes the first
  Player Turn that Season, then the next letter, and so on. This sequence is the
  **Command Order** (distinct from Turn Order).

### Wrong-Season cards & non-cards
- An Action card for the **wrong Season** still has an **Emergency Command** value
  (§10.11) but cannot otherwise be used for a normal Command.
- A player **not playing any Action card** that Season has **no Player Turn** —
  they cannot move units or resolve Battles (§12.1).
- **Investment cards** have **no Command Value** (except to bluff the Command
  ability) — discard them when revealed.

## 10.11 Emergency Command

An Action card for the wrong Season has an **Emergency Command value of two**
(**four** for the Axis). Units activated by Emergency Command:

- can make **minimal unit movement** (including into Disputed areas), **but**
- **no new Battles** can be started, and there is **no Combat Phase** that Player
  Turn.

If the **Command Priority letter** cited on the card still applies it is used for
ordering (if tied, the opponent chooses the Command Order).

---

## What the Command Phase sets up

After the Command Phase, **Player Turns** are taken in Command Order. Each Player
Turn consists of:

1. **Movement Phase** (§11.0) — see `05 - Movement Phase.md`
2. **Combat Phase** (§12.0) — see `06 - Combat Phase.md`

The number of units a player may move equals the **Command Value** of the card
they played. (The **Commands Available** marker can track this on the Production
track, moved down as each unit is moved.)

---

## Winter Command Phase (USSR only, §10.4)

In **Winter**, only the **USSR** may play **one** Command card (for *any*
Season) to move units and fight Battles **within the USSR only**. A Winter Supply
Phase follows, applying to USSR units only.
